Aplicativo desenvolvido para o Teste de Desenvolvimento Unity da Antidoto Design.

O tipo e aplicativo escolhido foi Quiz, foi projetado de acordo com as especificações do
Figma, por isso possui um fluxo contínuo e uma interface muito intuitiva que dispensa
explicações. De acordo com o Figma a orientação do aplicativo deve ser em retrato, que é
compatível com dispositivos mobile, contudo o teste exige que o Quiz seja compilado para
Windows, por isso a orientação foi adaptada para "retrato dentro de paisagem" com um espaço
vazio. O código foi comentado para o melhor desfrutar de sua coerência e organização.

A pasta "Build" possui a versão compilada com um executável, a mesma versão que se encontra
na pasta "AntidotoDesignChallenge\Builds\Versão 1.0" .
